# 🚀 Quick Start — Landing Page React

## Instalação Expressa

```bash
# 1. Navegue até a pasta web
cd web

# 2. Instale as dependências
npm install

# 3. Rode o projeto
npm run dev
```

🎉 **Pronto!** Acesse [http://localhost:3000](http://localhost:3000)

## 📦 O que foi instalado?

- **React 18** + **Vite 5** (super rápido!)
- **Tailwind CSS 3.4** (framework de utilitários)
- **lucide-react** (ícones modernos)

**Tamanho:** ~250MB de `node_modules` (padrão para projetos React)

## 🎨 Features Visuais

✅ Tipografia extrema com tracking negativo  
✅ Glassmorfismo e blur effects  
✅ Animações CSS customizadas  
✅ Dark mode disruptivo na seção revelação  
✅ Bento box layout assimétrico  
✅ Glow effects com cores neon  
✅ 100% responsivo  

## 🛠️ Comandos Úteis

```bash
npm run dev      # Desenvolvimento (port 3000)
npm run build    # Build de produção
npm run preview  # Preview do build
```

## 📱 Compatibilidade

- ✅ Chrome, Firefox, Safari, Edge (últimas versões)
- ✅ Mobile (iOS Safari, Chrome Android)
- ✅ Tablets e desktops
